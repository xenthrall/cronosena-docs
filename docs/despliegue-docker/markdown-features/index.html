Características de la documentación

La documentación de CronoSENA aprovecha el poder de Markdown junto con funcionalidades extendidas para ofrecer una experiencia de lectura clara, interactiva y moderna.

Front Matter​

Cada documento Markdown contiene metadatos en la parte superior llamados Front Matter, que ayudan a organizar la documentación de CronoSENA:

ejemplo.md
---
id: ejemplo-id
title: Título del documento
description: Breve descripción de este documento
slug: /mi-url-personalizada
---

## Encabezado Markdown

¡Markdown de ejemplo para la documentación de CronoSENA!

Enlaces​

Puedes encontrar referencias internas y externas fácilmente usando enlaces en Markdown, ideal para navegar entre las diferentes secciones del sistema y la documentación técnica.

Conoce cómo [levantar el entorno local](/despliegue/entorno-local).

Imágenes​

Puedes incluir imágenes para ilustrar funcionalidades de CronoSENA, diagramas de flujo, capturas de pantalla de la plataforma o ejemplos visuales.

![Logo CronoSENA](/img/crono-mountains.svg)

Asegúrate de ubicar imágenes en la carpeta static/img para rutas absolutas, o cerca del archivo para rutas relativas.
