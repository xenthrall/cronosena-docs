# Cómo correr el proyecto CronoSENA en local?
## Visita nuestro repositorio y descarga el archivo .ZIP

## O clona directamente el repositorio

```bash
git clone [https://github.com/xenthrall/CronoSENA.git]
```

# Requisitos

Antes de desplegar **CronoSENA** de manera local, asegúrate de contar con las siguientes herramientas instaladas en tu sistema:

## Herramientas necesarias

- **Git**: Para clonar el repositorio oficial de CronoSENA.


## Instalación de herramientas

- **Git**: Descárgalo e instálalo desde [git-scm.com](https://git-scm.com/downloads).
- **visual studio code**: Descárgalo e instálalo desde [code.visualstudio.com](https://code.visualstudio.com/)


> Asegúrate de que todas las herramientas estén correctamente instaladas y configuradas antes de continuar con el despliegue del sistema.
